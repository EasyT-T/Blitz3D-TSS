ParseDomainTXT txt$, name$

Parameters

txt = TXT answer of domain
name = name of the value in TXT

Description

Get value from TXT answer, use for feature such as check update.

If the last character of string isn't semicolon(;), then last value will get error in parsing(name of value will contain in value).

See Also: GetDomainTXT

Example

txt$ = GetDomainTXT("version.scpcbgame.cn") ; get txt answer
Print ParseDomainTXT(txt, "version") ; parse and get value of "version"
Print ParseDomainTXT(txt, "date") ; parse and get value of "date"
WaitKey
End
