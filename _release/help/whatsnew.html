What's New

Some differences with vanilla Blitz3D

IDE

  • No Backup: Backup system is pretty annoying, you can disable backup in “Edit”->“No Backup?”.
  • Modern toolbar icons: New icons!

Compiler

  • UTF-8 Support: Now Blitz3D can display asian character without userlib such as FastText or SpeedText.
  • ANSI Support: Text's encode from system may be ANSI, now you can display them.
    For Example: Text x, y, GfxDriverName(1), 0, 0, 1（x, y, txt$, xPos%=0, yPos%=0, encoding%=0）
  • More keywords: Interger=Int, String=Str.
  • Spare Function Name: Now functions have there spare name: Blitz_functionName
