What's New

Some differences with vanilla Blitz3D

IDE

  • No Backup: Backup system is pretty annoying, you can disable backup in “Edit”->“Need Backup?”.
  • Drag to open: Drag files into Blitz3d IDE to open!
  • Modern toolbar icons: New icons!

Compiler

  • UTF-8 Support: Now Blitz3D can display asian character without userlib such as FastText or SpeedText.
  • ANSI Support: Now you can use ConvertToANSI and ConvertToUTF8 to fix some bugs.
    For Example: Text x, y, GfxDriverName(1), 0, 0, 1（x, y, txt$, xPos%=0, yPos%=0, encoding%=0）
  • More keywords: Interger=Int, Switch=Select... and more!
  • Spare Function Name: Now functions have there spare name: Blitz_functionName
  • Get domain's TXT answer: Use GetDomainTXT and SpareDomainTXT to make a update checker!
    Note: you can't make a updater by these two functions. In fact, download update pack and install it is very complex and easy to encounter bugs.
